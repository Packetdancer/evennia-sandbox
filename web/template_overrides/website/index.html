Pax's Evennia Sandbox

This is a basic server put together atop Evennia, your friendly neighborhood next-generation MUD development system and server.

This particular Evennia installation is designed to provide an experience roughly familiar to MUSH/MUX users, while still being forward-looking in terms of web presence and extensibility.

You can log into this website with your game user and password; when logged in, you'll see some of the on-game resources available, such as the game's bulletin boards exposed as a web forum.

{% if webclient_enabled %}

You can even connect to the game directly from your browser using our online client!

{% endif %}

Some of the notable features in this particular Evennia database are:

  • A bboard system with web integration and threaded conversations, as well as Markdown support.
  • A standardized display and notification system, to make it easy to configure the game to your desired appearance.
  • Standard MUSH-like commands, such as finger and where.

Future planned improvements include:

  • An events calendar for scheduling on-game events, accessible from the website and able to be imported into calendar tools.
  • A mail system with web integration.
  • A jobs system with web integration.

If you'd like to fork or contribute to this particular Evennia setup, you can find the code on the project's Github repository.

Evennia

Evennia is an open-source MUD server built in Python, on top of the Twisted and Django frameworks. This combination of technologies allows for the quick and easy creation of the game of your dreams - as simple or as complex as you like.
